# Retaped
the revolt.chat client retaped from the one held together by duct tape and bad code

## Features include
- replies(send & recieve)
- sending embeds
- masquarade rendering
- channel list
- server list
- dm list
- image rendering
- automatic username loading
- automatic message loading
- username cache
- channel info
- themes
- side-by-side layout
- file uploading(in theory)
